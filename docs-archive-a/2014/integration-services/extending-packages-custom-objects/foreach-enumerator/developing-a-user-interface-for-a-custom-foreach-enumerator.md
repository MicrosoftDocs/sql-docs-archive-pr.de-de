# <a name="developing-a-user-interface-for-a-custom-foreach-enumerator"></a><span data-ttu-id="77039-102">Entwickeln einer Benutzeroberfläche für einen benutzerdefinierten ForEach-Enumerator</span><span class="sxs-lookup"><span data-stu-id="77039-102">Developing a User Interface for a Custom ForEach Enumerator</span></span>
  <span data-ttu-id="77039-103">Nachdem Sie die Implementierung der Eigenschaften und Methoden der Basisklasse überschrieben haben, um benutzerdefinierte Funktionen bereitzustellen, möchten Sie vielleicht eine benutzerdefinierte Benutzeroberfläche für den Foreach-Enumerator erstellen.</span><span class="sxs-lookup"><span data-stu-id="77039-103">After you have overridden the implementation of the properties and methods of the base class to provide your custom functionality, you may want to create a custom user interface for your Foreach enumerator.</span></span> <span data-ttu-id="77039-104">Wenn Sie keine individuelle Benutzeroberfläche erstellen, können die Benutzer den neuen benutzerdefinierten ForEach-Enumerator nur über das Eigenschaftenfenster konfigurieren.</span><span class="sxs-lookup"><span data-stu-id="77039-104">If you do not create a custom user interface, users can only configure the new custom Foreach enumerator by using the Properties window.</span></span>  
  
 <span data-ttu-id="77039-105">Erstellen Sie in einem benutzerdefinierten Benutzeroberflächenprojekt oder einer entsprechenden Assembly eine Klasse, die den <xref:Microsoft.SqlServer.Dts.Runtime.ForEachEnumeratorUI> implementiert.</span><span class="sxs-lookup"><span data-stu-id="77039-105">In a custom user interface project or assembly, you create a class that implements <xref:Microsoft.SqlServer.Dts.Runtime.ForEachEnumeratorUI>.</span></span> <span data-ttu-id="77039-106">Diese Klasse wird von „System.Windows.Forms.UserControl“ abgeleitet, das typischerweise zur Erstellung eines zusammengesetzten Steuerelements als Host für andere Windows Forms-Steuerelemente verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="77039-106">This class derives from System.Windows.Forms.UserControl, which is typically used to create a composite control to host other Windows Forms controls.</span></span> <span data-ttu-id="77039-107">Das von Ihnen erstellte Steuerelement wird im Bereich **Enumeratorkonfiguration** der Registerkarte **Auflistung** des **Foreach-Schleifen-Editors** angezeigt.</span><span class="sxs-lookup"><span data-stu-id="77039-107">The control that you create is displayed in the **Enumerator configuration** area of the **Collection** tab of the **Foreach Loop Editor**.</span></span>  
  
> [!IMPORTANT]  
>  <span data-ttu-id="77039-108">Nach dem Signieren und Erstellen der benutzerdefinierten Benutzeroberfläche und der Installation im globalen Assemblycache wie in [Building, Deploying, and Debugging Custom Objects](../building-deploying-and-debugging-custom-objects.md) (Erstellen, Bereitstellen und Debuggen von benutzerdefinierten Objekten) beschrieben müssen Sie den vollqualifizierten Namen dieser Klasse in der <xref:Microsoft.SqlServer.Dts.Runtime.DtsForEachEnumeratorAttribute.UITypeName%2A>-Eigenschaft des <xref:Microsoft.SqlServer.Dts.Runtime.DtsForEachEnumeratorAttribute> bereitstellen.</span><span class="sxs-lookup"><span data-stu-id="77039-108">After signing and building your custom user interface and installing it in the global assembly cache as described in [Building, Deploying, and Debugging Custom Objects](../building-deploying-and-debugging-custom-objects.md), remember to provide the fully qualified name of this class in the <xref:Microsoft.SqlServer.Dts.Runtime.DtsForEachEnumeratorAttribute.UITypeName%2A> property of the <xref:Microsoft.SqlServer.Dts.Runtime.DtsForEachEnumeratorAttribute>.</span></span>

### <a name="initializing-the-user-interface"></a><span data-ttu-id="77039-110">Initialisieren der Benutzeroberfläche</span><span class="sxs-lookup"><span data-stu-id="77039-110">Initializing the User Interface</span></span>  
 <span data-ttu-id="77039-111">Sie überschreiben die <xref:Microsoft.SqlServer.Dts.Runtime.ForEachEnumeratorUI.Initialize%2A>-Methode, um Verweise auf das Hostobjekt sowie die im Paket definierten Auflistungen von Verbindungs-Managern und Variablen zwischenzuspeichern.</span><span class="sxs-lookup"><span data-stu-id="77039-111">You override the <xref:Microsoft.SqlServer.Dts.Runtime.ForEachEnumeratorUI.Initialize%2A> method to cache references to the host object, and to the collections of connection managers and variables defined in the package.</span></span>  
  
### <a name="setting-properties-on-the-user-interface-control"></a><span data-ttu-id="77039-112">Festlegen von Eigenschaften für das Benutzeroberflächensteuerelement</span><span class="sxs-lookup"><span data-stu-id="77039-112">Setting Properties on the User Interface Control</span></span>  
 <span data-ttu-id="77039-113">Die UserControl-Klasse, von der die Benutzeroberflächenklasse abgeleitet wird, dient zur Verwendung als zusammengesetztes Steuerelement zum Hosten anderer Windows Forms-Steuerelemente.</span><span class="sxs-lookup"><span data-stu-id="77039-113">The UserControl class, from which the user interface class is derived, is intended for use as a composite control to host other Windows Forms controls.</span></span> <span data-ttu-id="77039-114">Da diese Klasse als Host für andere Steuerelemente dient, können Sie Ihre benutzerdefinierte Oberfläche durch Ziehen und Ablegen von Steuerelementen entwerfen. Sie können diese wie bei jeder Windows Forms-Anwendung anordnen, ihre Eigenschaften festlegen und zur Ausführungszeit auf ihre Ereignisse reagieren.</span><span class="sxs-lookup"><span data-stu-id="77039-114">Because this class hosts other controls, you can design your custom user interface by dragging and dropping controls, arranging them, setting their properties, and responding at run time to their events as in any Windows Forms application.</span></span>  
  
### <a name="saving-settings"></a><span data-ttu-id="77039-115">Speichern von Einstellungen</span><span class="sxs-lookup"><span data-stu-id="77039-115">Saving Settings</span></span>  
 <span data-ttu-id="77039-116">Sie überschreiben die <xref:Microsoft.SqlServer.Dts.Runtime.ForEachEnumeratorUI.SaveSettings%2A>-Methode, um die vom Benutzer festgelegten Werte beim Beenden des Editors aus den Steuerelementen in die Eigenschaften des Enumerators zu kopieren.</span><span class="sxs-lookup"><span data-stu-id="77039-116">You override the <xref:Microsoft.SqlServer.Dts.Runtime.ForEachEnumeratorUI.SaveSettings%2A> method to copy the values selected by the user from the controls to the properties of the enumerator when the user closes the editor.</span></span>
