## <a name="subselect-statement"></a><span data-ttu-id="5883f-407">Unterauswahlanweisung</span><span class="sxs-lookup"><span data-stu-id="5883f-407">Subselect statement</span></span>

```  
[WITH [<calc-clause> ...]]

[[CELL] PROPERTIES <cellprop> [, <cellprop> ...]]
```  
