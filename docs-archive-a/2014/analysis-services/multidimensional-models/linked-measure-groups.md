## <a name="usage-limitations"></a><span data-ttu-id="94c47-110">Nutzungseinschränkungen</span><span class="sxs-lookup"><span data-stu-id="94c47-110">Usage Limitations</span></span>  
 <span data-ttu-id="94c47-111">Wie bereits erwähnt, besteht eine wichtige Einschränkung bei der Verwendung von verknüpften Measures darin, dass ein verknüpftes Measure nicht direkt angepasst werden kann.</span><span class="sxs-lookup"><span data-stu-id="94c47-111">As noted previously, an important constraint to using linked measures is an inability to customize a linked measure directly.</span></span> <span data-ttu-id="94c47-112">Änderungen an Datentyp, Format, Datenbindung und Sichtbarkeit sowie an der Mitgliedschaft der Elemente in der Measuregruppe selbst sind Änderungen, die an der ursprünglichen Measuregruppe vorgenommen werden müssen.</span><span class="sxs-lookup"><span data-stu-id="94c47-112">Modifications to the data type, format, data binding, and visibility, as well as membership of the items in the measure group itself, are all changes that must be made in the original measure group.</span></span>  
  
 <span data-ttu-id="94c47-113">Funktionell sind verknüpfte Measuregruppen identisch mit anderen Measuregruppen, wenn Clientanwendungen darauf zugreifen, und sie werden auf dieselbe Weise wie andere Measuregruppen abgefragt.</span><span class="sxs-lookup"><span data-stu-id="94c47-113">Operationally, linked measure groups are identical to other measure groups when accessed by client applications, and are queried in the same manner as other measure groups.</span></span>

 <span data-ttu-id="94c47-117">In der folgenden Liste sind Nutzungseinschränkungen zusammengefasst.</span><span class="sxs-lookup"><span data-stu-id="94c47-117">The following list summarizes usage limitations.</span></span>

-   <span data-ttu-id="94c47-119">Sie können in einer verknüpften Measuregruppe keine Measures hinzufügen oder entfernen.</span><span class="sxs-lookup"><span data-stu-id="94c47-119">You cannot add or remove measures in a linked measure group.</span></span> <span data-ttu-id="94c47-120">Die Mitgliedschaft wird nur in der ursprünglichen Measuregruppe definiert.</span><span class="sxs-lookup"><span data-stu-id="94c47-120">Membership is defined only in the original measure group.</span></span>  
  
-   <span data-ttu-id="94c47-121">Das Rückschreiben wird in verknüpften Measuregruppen nicht unterstützt.</span><span class="sxs-lookup"><span data-stu-id="94c47-121">Writeback is not supported in linked measure groups.</span></span>  
  
-   <span data-ttu-id="94c47-122">Verknüpfte Measuregruppen können nicht in mehreren m:n-Beziehungen verwendet werden, insbesondere wenn sich diese Beziehungen in verschiedenen Cubes befinden.</span><span class="sxs-lookup"><span data-stu-id="94c47-122">Linked measure groups cannot be used in multiple many-to-many relationships, especially when those relationships are in different cubes.</span></span> <span data-ttu-id="94c47-123">Andernfalls können mehrdeutige Aggregationen entstehen.</span><span class="sxs-lookup"><span data-stu-id="94c47-123">Doing so can result in ambiguous aggregations.</span></span> <span data-ttu-id="94c47-124">Weitere Informationen finden Sie unter [Falsche Mengen für verknüpfte Measures in Cubes mit m:n-Beziehungen](https://social.technet.microsoft.com/wiki/contents/articles/22911.incorrect-amounts-for-linked-measures-in-cubes-containing-many-to-many-relationships-ssas-troubleshooting.aspx).</span><span class="sxs-lookup"><span data-stu-id="94c47-124">For more information, see [Incorrect Amounts for Linked Measures in Cubes containing Many-to-Many Relationships](https://social.technet.microsoft.com/wiki/contents/articles/22911.incorrect-amounts-for-linked-measures-in-cubes-containing-many-to-many-relationships-ssas-troubleshooting.aspx).</span></span>  
  
 <span data-ttu-id="94c47-125">Die in einer verknüpften Measuregruppe enthaltenen Measures können nur anhand von verknüpften Dimensionen direkt organisiert werden, die aus derselben [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)] -Datenbank abgerufen wurden.</span><span class="sxs-lookup"><span data-stu-id="94c47-125">The measures contained in a linked measure group can be directly organized only along linked dimensions retrieved from the same [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)] database.</span></span> <span data-ttu-id="94c47-126">Mithilfe von berechneten Elementen können Sie jedoch Informationen aus verknüpften Measuregruppen mit anderen, nicht verknüpften Dimensionen im Cube verbinden.</span><span class="sxs-lookup"><span data-stu-id="94c47-126">However, you can use calculated members to relate information from linked measure groups to the other, non-linked dimensions in your cube.</span></span> <span data-ttu-id="94c47-127">Sie können auch eine indirekte Beziehung wie einen Verweis oder eine m:n-Beziehung verwenden, um nicht verknüpfte Dimensionen mit einer verknüpften Measuregruppe zu verbinden.</span><span class="sxs-lookup"><span data-stu-id="94c47-127">You can also use an indirect relationship, such as a reference or many-to-many relationship, to relate non-linked dimensions to a linked measure group.</span></span>
