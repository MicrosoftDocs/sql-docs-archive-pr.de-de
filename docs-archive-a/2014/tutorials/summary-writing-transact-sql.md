# <a name="summary-writing-transact-sql"></a>Zusammenfassung: Schreiben von Transact-SQL-Code
  Der Abfrage-Editor ist ein wichtiger Schritt in Bezug auf eine anspruchsvollere Erstellungs Umgebung in der [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] Entwicklung. Sie können den Abfrage-Editor problemlos in den Vollbildmodus schalten, um Ihren Code anzuzeigen, oder schnell eine Verbindung zu einem anderen Server herstellen. Zusätzliche Funktionen finden Sie u. a. im Dialogfeld **Optionen** , auf das Sie über das Menü **Extras** zugreifen.
