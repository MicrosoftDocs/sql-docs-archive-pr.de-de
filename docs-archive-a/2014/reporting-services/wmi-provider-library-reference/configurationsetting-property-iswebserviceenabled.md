# <a name="iswebserviceenabled-property-wmi-msreportserver_configurationsetting"></a>IsWebServiceEnabled-Eigenschaft (WMI: MSReportServer_ConfigurationSetting)
  Gibt an, ob der Report Server-Webdienst aktiviert ist. Schreibgeschützt.  
  
## <a name="syntax"></a>Syntax  
  
```vb  
Public Dim IsWebServiceEnabled As Boolean  
```  
  
```csharp  
public boolean IsWebServiceEnabled;  
```  
  
## <a name="property-values"></a>Eigenschaftswerte  
 Ein schreibgeschützter `Boolean`-Wert. Der Wert `true` gibt an, dass der Report Server-Webdienst aktiviert ist.
