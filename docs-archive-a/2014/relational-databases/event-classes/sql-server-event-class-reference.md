# <a name="sql-server-event-class-reference"></a>Ereignisklassen in SQL Server – Referenz
  Mithilfe von [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)] können Sie Ereignisse aufzeichnen, sobald diese in einer Instanz von [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssDEnoversion](../../includes/ssdenoversion-md.md)] auftreten. Die erfassten Ereignisse sind Instanzen der Ereignisklassen in der Ablaufverfolgungsdefinition. Ereignisklassen und die dazugehörigen Ereigniskategorien sind in [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)]im Dialogfeld **Eigenschaften der Ablaufverfolgungsdatei** auf der Registerkarte **Ereignisauswahl** verfügbar.
