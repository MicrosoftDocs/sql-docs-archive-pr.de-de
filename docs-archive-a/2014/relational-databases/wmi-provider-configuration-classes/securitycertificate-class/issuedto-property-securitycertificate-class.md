## <a name="syntax"></a>Syntax  
  
```  
  
object  
.IssuedTo [= value]  
```  
  
## <a name="parts"></a>Bestandteile  
 *object*  
 Ein [SecurityCertificate-Klassenobjekt](securitycertificate-class.md) , das ein Sicherheitszertifikat darstellt.
